## Quick Deploy Commands:

### For Surge.sh:
```bash
npm install -g surge
cd your-project-folder
surge --domain kers0neserverclone.surge.sh
```

### For Netlify:
```bash
npm install -g netlify-cli
netlify deploy --prod --dir .
```

**Recommendation**: Use Netlify for easiest setup!
